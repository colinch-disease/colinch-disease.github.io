# Claude 사용법 실전 — 이메일, 보고서, 긴 글 요약 어떻게 쓰나

![AI 글쓰기](https://images.unsplash.com/photo-1455390582262-044cdead277a?w=800&q=80)

ChatGPT를 오래 쓰다 Claude를 처음 써봤을 때 첫 반응은 "비슷하네"였다. 근데 한 달 쓰고 나니 분명히 다른 부분이 있었다. 특히 글쓰기 품질과 긴 문서를 다루는 능력에서.

## ChatGPT와 뭐가 다른가

가장 큰 차이는 **긴 문서 처리 용량**이다. 계약서 20페이지, 보고서 전체를 한꺼번에 붙여 넣고 "핵심만 요약해줘"가 가능하다. ChatGPT 무료 버전은 이게 잘 안 된다.

그리고 글 톤이 조금 다르다. ChatGPT는 조금 더 직설적이고, Claude는 문맥을 더 잘 이해하고 자연스럽게 쓰는 느낌이다. 글쓰기 작업에서 특히 차이가 보인다.

## 실전 1 — 계약서·보고서 요약

계약서를 받았는데 다 읽기 시간이 없을 때. 전문을 붙여 넣고 이렇게 묻는다.

**프롬프트:**
```
아래 계약서를 읽고 다음 항목으로 요약해줘:
1. 핵심 계약 조건 (금액, 기간, 납품 조건)
2. 내가 주의해야 할 불리한 조항
3. 애매하거나 확인이 필요한 부분

[계약서 전문 붙여넣기]
```

법적 판단은 변호사에게 맡겨야 하지만, 뭘 확인해야 하는지 파악하는 속도가 훨씬 빨라진다.

## 실전 2 — 이메일 작성

받은 이메일에 답장을 써야 하는데 어떻게 시작해야 할지 모를 때.

**프롬프트:**
```
아래 이메일에 답장을 써줘.
상황: 미팅 요청을 정중히 거절하되, 다음 달 재조율 가능성은 열어두고 싶다.
톤: 정중하고 따뜻하게, 너무 딱딱하지 않게.

[받은 이메일 붙여넣기]
```

한 번에 완벽한 답장이 나오진 않는데, "좀 더 짧게", "마지막 문단을 더 구체적으로" 식으로 수정 요청하면서 다듬으면 된다.

## 실전 3 — 긴 글·뉴스 요약

업계 뉴스, 긴 아티클, 백서를 빠르게 파악해야 할 때.

**프롬프트:**
```
아래 글을 읽고:
1. 핵심 주장 3줄 요약
2. 내가 실무에서 참고할 만한 내용
3. 의문이 드는 부분이나 추가 확인 필요한 것

[긴 글 붙여넣기]
```

## 아쉬운 점

실시간 정보가 없다. "오늘 환율"이나 "최근 뉴스"는 모른다. 그런 정보가 필요하면 Perplexity를 따로 써야 한다. 또 이미지 생성은 안 된다.
