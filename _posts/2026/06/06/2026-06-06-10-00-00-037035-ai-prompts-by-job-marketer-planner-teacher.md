# AI 프롬프트 직업별 모음 — 마케터, 기획자, 교사, HR 담당자용

![직업별 AI 활용](https://images.unsplash.com/photo-1522202176988-66273c2fd55f?w=800&q=80)

"AI 쓰면 좋다는데 뭘 어떻게 써야 하는지 모르겠다"는 말을 자주 듣는다. 직업마다 반복하는 작업이 다르고, 거기에 맞는 프롬프트가 따로 있다. 내 직업에 맞는 것만 가져다 쓰면 된다.

## 마케터용

**SNS 카피 작성:**
```
[상품: 여름 자외선차단 선크림, 타겟: 20대 여성]
인스타그램 피드용 카피 5가지 써줘.
조건: 2~3문장, 감성적인 톤, 이모지 포함, 해시태그 5개도 같이.
```

**경쟁사 분석 정리:**
```
[경쟁사 A, B, C의 SNS 게시물 내용 붙여넣기]
이 세 브랜드의 콘텐츠 전략 차이를 표로 정리해줘.
우리가 차별화할 수 있는 포인트도 2~3가지 뽑아줘.
```

**광고 문구 A/B 테스트용:**
```
[제품명: 수면 보조 앱]
같은 메시지를 다른 방식으로 표현한 광고 문구 3쌍 써줘.
각 쌍은 같은 핵심 메시지를 감성형/이성형으로 나눠서.
```

## 기획자용

**회의 아젠다 작성:**
```
다음 주 [신규 서비스 론칭 전략 회의] 아젠다를 만들어줘.
참석자: 기획, 마케팅, 개발 각 1명
시간: 1시간
논의할 핵심 결정 사항: 론칭 일정, 홍보 채널, KPI 설정
```

**기획서 목차 구조화:**
```
[프로젝트: 사내 온보딩 프로세스 개선]
기획서 목차 잡아줘. 
경영진 설득용이고, 예산 승인이 목적이야.
```

## 교사용

**수업 활동 아이디어:**
```
[과목: 중학교 2학년 국어, 단원: 논설문 쓰기]
학생들이 흥미롭게 참여할 수 있는 토론 주제 5가지랑 활동 방법 알려줘.
수업 시간은 45분.
```

**학부모 안내문 작성:**
```
아래 내용을 학부모 안내문으로 써줘.
내용: 다음 달 수학여행 일정 변경 안내
톤: 정중하고 명확하게, 학부모가 불안해하지 않도록.
변경 사유: [사유 입력]
```

## HR 담당자용

**채용 공고 작성:**
```
[직무: 콘텐츠 마케터, 경력 3년 이상]
채용 공고 초안 써줘.
포함: 주요 업무, 자격 요건, 우대 사항, 근무 환경 및 복리후생.
톤: 지원자가 매력을 느낄 수 있게, 딱딱하지 않게.
```

**직원 설문 문항 작성:**
```
분기 직원 만족도 조사 문항 10개 만들어줘.
측정 항목: 업무 환경, 성장 기회, 팀워크, 복리후생 만족도.
형식: 5점 척도 + 주관식 각 2개씩.
```

---

프롬프트는 길게 쓸수록 좋다. 상황, 대상, 조건을 구체적으로 줄수록 원하는 결과가 나온다. 처음엔 어색하지만 두세 번 쓰다 보면 자기만의 패턴이 생긴다.
