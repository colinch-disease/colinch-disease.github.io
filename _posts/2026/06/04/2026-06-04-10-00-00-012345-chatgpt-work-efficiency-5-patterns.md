# ChatGPT 업무 활용법 — 실제 직장인이 쓰는 5가지 패턴

![직장인 업무](https://images.unsplash.com/photo-1497032628192-86f99bcd76bc?w=800&q=80)

ChatGPT를 "신기한 채팅 AI"로만 쓰다가 어느 순간 업무 도구로 쓰기 시작했다. 처음엔 효과가 있는지도 몰랐는데, 반년 쓰고 나니 없으면 불편한 도구가 됐다. 특히 반복적으로 쓰게 되는 패턴이 생겼다.

## 패턴 1 — 보고서 초안 잡기

보고서 쓰는 시간의 절반 이상이 "어떻게 시작하지"에서 소모된다. ChatGPT에게 목적과 청중만 알려주면 구조를 잡아준다.

**프롬프트:**
```
[주제: OO 사업 현황 보고]
대상: 임원진
목적: 1분기 실적과 2분기 방향 공유
형식: 헤드라인 + 핵심 수치 + 시사점 구조로 1페이지 분량 초안 작성해줘.
```

초안이 100% 완성형은 아닌데, 빈 페이지를 채우는 게 핵심이다. 초안이 있으면 수정이 훨씬 빠르다.

## 패턴 2 — 이메일 다듬기

쓰고 나서 "이 말투가 맞나?" 싶을 때 ChatGPT에 붙여 넣고 "비즈니스 이메일 톤으로 다듬어줘"라고 한다. 너무 딱딱하거나 너무 캐주얼한 게 순식간에 정리된다.

**프롬프트:**
```
아래 이메일을 정중하지만 친근한 비즈니스 톤으로 다듬어줘.
불필요한 중복 표현은 빼고, 핵심만 남겨줘.

[이메일 내용 붙여넣기]
```

## 패턴 3 — 회의 전 질문 리스트 뽑기

회의 주제를 알려주고 "어떤 걸 미리 파악해야 하나"를 물어본다. 놓치는 포인트가 줄어든다.

**프롬프트:**
```
내일 [신규 거래처 계약 조건 협의] 회의가 있다.
사전에 확인해야 할 사항과 회의에서 물어볼 질문 10개 뽑아줘.
```

## 패턴 4 — 낯선 분야 빠르게 파악하기

새 프로젝트에 던져졌을 때, 관련 용어와 맥락을 빠르게 파악하는 데 쓴다. 구글 검색보다 맥락 있는 설명이 나온다.

**프롬프트:**
```
나는 마케터인데 갑자기 ESG 보고서 작업을 맡았다.
ESG가 뭔지, 기업이 왜 신경 쓰는지, 보고서에 보통 어떤 내용이 들어가는지 쉽게 설명해줘.
```

## 패턴 5 — 아이디어 브레인스토밍

혼자 막혀 있을 때 5분짜리 브레인스토밍 상대로 쓴다. 10개 아이디어 중 1~2개만 건져도 충분하다.

**프롬프트:**
```
[20~30대 직장인 대상 사내 복지 이벤트 아이디어]
예산: 1인당 3만원 이내
조건: 비대면 참여 가능, 팀 결속에 도움
아이디어 10개만 뽑아줘.
```
