# Colinch 건강관리 경량 하네스

- 공통 규칙은 상위 `AGENTS.md`를 따른다. 자동 포스팅 중 `CLAUDE.md` 전체를 읽지 않는다.
- 언어: 한국어 직접체 `~다`. 글은 의학적 진단·치료 조언이 아니라 생활 관리 경험 정보다.
- 주제: 소화기, 대사·내분비, 근골격계·통증, 수면·정신건강, 피부·알레르기 생활 관리.
- 반드시 “진단과 치료는 의사와 상담해야 한다”는 취지의 문장을 자연스럽게 넣는다.
- 웹 검색: 최신 연구·가이드라인을 1~2회 확인하고 병원·학회·공공기관 등 신뢰 출처만 쓴다.
- SEO: 질환명, 증상, 생활 관리 방법을 title·description·첫 문단에 넣는다.
- 태그 후보: `역류성식도염`, `과민성대장증후군`, `위염`, `당뇨`, `고혈압`, `갑상선저하증`, `고지혈증`, `허리디스크`, `무릎관절염`, `족저근막염`, `불면증`, `아토피`, `알레르기비염`, `생활관리`, `식단관리`, `자가관리`.
- 시각 자료: 증상별 체크리스트, 생활습관 전후 비교표, 병원 방문 기준 표, 하루 루틴 표를 우선한다.
- “낫는다”, “치료된다”처럼 효과를 단정하지 않는다. 위험 신호와 병원 방문 기준을 포함한다.
